# Idle Clicker - Яндекс Игры 🎮

Полнофункциональная idle clicker игра с монетизацией для платформы Яндекс.Игры.

## 🎮 Основные возможности

### Игровой процесс
- ✅ **Клики** - базовый источник монет (улучшаемый)
- ✅ **Автоматический доход** - система рабочих (DPS)
- ✅ **Улучшения** - повышение урона и эффективности
- ✅ **Рабочие** - пассивный заработок (муравей, пчела, паук, дракон)
- ✅ **Сохранение** - автосохранение каждые 30 секунд + ручное сохранение

### Монетизация
- 💰 **Премиум валюта** - звезды (⭐)
- 🛍️ **Магазин** - пакеты премиум монет (99, 499, 1999 рублей)
- 📊 **Аналитика** - интеграция с Яндекс SDK
- 📈 **Monetization-friendly** - легко интегрировать платежи

## 🚀 Быстрый старт

```bash
# Клонируем репозиторий
git clone https://github.com/bobah19890501/idle-clicker.git
cd idle-clicker

# Запустите локальный сервер
python -m http.server 8000
# или
npx http-server

# Откройте http://localhost:8000
```

# Развертывание на Яндекс.Игры 🚀

## 📋 Требования

1. ✅ GitHub аккаунт (есть)
2. ✅ Яндекс ID (бесплатно)
3. ✅ GitHub Pages (бесплатно)

## 📤 Шаг 1: Включение GitHub Pages

1. Откройте репозиторий: https://github.com/bobah19890501/idle-clicker
2. Перейдите в **Settings** (⚙️)
3. В левом меню выберите **Pages**
4. В разделе "Source":
   - Branch: `main`
   - Folder: `/ (root)`
5. Нажмите **Save**

**Ваша игра будет доступна по адресу:**
```
https://bobah19890501.github.io/idle-clicker/
```

## 🎮 Шаг 2: Регистрация на Яндекс.Игры

1. Перейдите на: https://developer.yandex.ru/
2. Авторизуйтесь с Яндекс ID
3. Согласитесь с условиями
4. Заполните профиль разработчика

## 🎯 Шаг 3: Создание игры

1. В консоли нажмите **Создать игру**
2. Заполните информацию:
   - **Название**: Idle Clicker
   - **Описание**: Увлекательная idle clicker игра с автоматическим заработком
   - **Жанр**: Casual / Simulator
   - **Язык**: Русский
3. Нажмите **Далее**

## ⚙️ Шаг 4: Настройка параметров

### Основные параметры:
- **URL игры**: `https://bobah19890501.github.io/idle-clicker/`
- **Размер окна**: На весь экран (fullscreen)
- **Ориентация**: Портрет (portrait)

## 💳 Шаг 5: Монетизация

### Включение платежей

1. Перейдите в **Монетизация**
2. Включите **In-App Purchases**
3. Добавьте товары:

```
1. premiumPack1
   Название: Стартовый пакет
   Цена: 99 RUB

2. premiumPack2
   Название: Золотой пакет
   Цена: 499 RUB

3. premiumPack3
   Название: Платиновый пакет
   Цена: 1999 RUB
```

## ✅ Шаг 6: Отправка на модерацию

1. Нажмите **Отправить на модерацию**
2. Подтвердите соответствие правилам
3. Отправьте на проверку

**Модерация: 3-7 дней**

## 🎉 Шаг 7: Публикация

После одобрения:

1. Нажмите **Опубликовать**
2. Выберите страны
3. Подтвердите публикацию

**Поздравляем! Ваша игра готова к публикации! 🚀**
